# Publishing to GitHub Marketplace

This repo is ready for Marketplace listing. One-time org steps:

1. Open https://github.com/UnplugAI/unplug-scan-action/releases/tag/v1.0.1
2. Click **Publish this Action to GitHub Marketplace** (org admin)
3. Accept the GitHub Marketplace Developer Agreement
4. Category: **Code quality** or **Security**
5. Set logo (optional): Unplug shield branding

Consumers install with:

```yaml
- uses: UnplugAI/unplug-scan-action@v1
```

Major tag `v1` tracks the latest `v1.x` release.
